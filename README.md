# WAMP Multi-Country Dashboard

Static HTML dashboard for the AB InBev Weighted Average Market Price (WAMP) analysis,
covering 11 countries (Poland, Thailand, Austria, Denmark, Norway, Switzerland,
New Zealand, Costa Rica, Romania, Venezuela, Puerto Rico).

## Live URL

After Vercel imports this repo: `https://abi-wamp-multi.vercel.app/`

## What's in here

- `index.html` — the full dashboard, self-contained (data embedded as JSON, no external API calls).
- `vercel.json` — Vercel config (clean URLs, security headers).
- `.gitignore` — excludes source workbooks, scripts, and build artefacts.

## How it's built

The HTML is generated by the `wamp-multi-country` Cursor skill from the per-country
Nielsen / Circana source workbooks. The build script lives outside this repo at
`C:\Users\99808455\.cursor\skills\wamp-multi-country\scripts\build_dashboard_multi.py`
and runs an ETL adapter per country before rendering the dashboard template.

## How to update the deployed site

1. Re-run the build script: `py build_dashboard_multi.py`
2. Copy the regenerated `wamp_dashboard_multi.html` into this folder as `index.html`
3. `git add index.html && git commit -m "..." && git push`
4. Vercel auto-deploys in ~30 seconds.
